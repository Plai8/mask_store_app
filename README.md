# 😷 Mask Store App

Mask Store App 是一個 **純前端靜態網站**，可查詢使用者所在地區附近的藥局口罩庫存與營業時間。  
本專案以 **Vue.js** 建構，並使用 **Leaflet** 結合台灣藥局 API 進行地圖與庫存資訊呈現。

## 🌍 Demo 連結  
👉 [Mask Store App 線上預覽](https://plai8.github.io/mask_store_app/)

## 🔥 功能介紹
- 顯示使用者選取地區的藥局口罩庫存與詳細資訊（地址、營業資訊等）
- 使用 **Leaflet** 在地圖上呈現全台藥局位置
- 提供搜尋與過濾功能，提升使用者查詢效率
- 完整的 **RWD 響應式設計**，支援手機、平板與桌機使用

## 🛠 技術棧
- **Frontend:** HTML  
- **Style:** CSS / Tailwind CSS  
- **Framework:** Vue.js  
- **State Management:** Pinia  
- **Map & API:** Leaflet + 台灣藥局公開資料 API  
- **Tools:** GitHub（使用 GitHub Pages 部署）
